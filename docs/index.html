Features

Everything you need to slap.

Built-in accelerometer detection, multiple sound packs, customizable key bindings, and a mode that would make Tony Stark proud.

⚡

Accelerometer Detection

Reads your MacBook's built-in IMU at 800Hz via IOKit HID. Detects impacts with peak-tracking and STA/LTA seismology algorithms.

🎵

6 Sound Packs

Whip Crack, Slap, Punch, Air Horn, Sussy UwU, and Custom. Each with a preview button. Volume and pitch scale with impact force.

⚡

Tony Stark Mode

Activate J.A.R.V.I.S. sounds. Double-clap opens Terminal with Claude Code and plays Iron Man soundtrack. Arc reactor UI included.

⌨

Custom Key Binds

Map any key to trigger on slap. Presets for 1, 2, 3, Space, Return, or record any custom key. Simulated via CGEvent.

🎧

Force-Scaled Audio

Harder slaps play louder with higher pitch. Logarithmic volume scaling and dynamic pitch shifting via AVAudioEngine.

📈

Live Dashboard

Real-time slap counter, force meter, recent activity feed, animated impact visualization with ripple effects.

Get Started

Install in 30 seconds.

Clone, build, slap. That's it.

# Clone the repo
git clone https://github.com/yusuf2145/slapsound.git
cd slapsound

# Build
swift build -c release

# Run (needs sudo for accelerometer)
sudo .build/release/SlapSound
