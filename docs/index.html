Whip cracks. Face slaps. Air horns. Whatever you want. Every slap also triggers a configurable key press.

FAQ

Why does it need sudo?
The MacBook's accelerometer is accessed via IOKit HID, which requires root privileges. The app only reads sensor data locally — nothing leaves your machine.
Which Macs are supported?
Apple Silicon MacBooks with M1 Pro or newer. These have a built-in IMU (Bosch BMI286) accelerometer. The base M1, Mac Mini, Mac Studio, and Intel Macs don't have it.
Will it damage my MacBook?
The accelerometer detects vibration — you don't need to hit hard. A firm tap on the side or trackpad area is enough. But also, slap responsibly.
Can I add my own sounds?
Yes. Drop any .mp3, .wav, or .m4a file into Sources/SlapSound/Resources/Sounds/ and rebuild. The app loads all sound files at startup.
The key press isn't working?
Go to System Settings > Privacy & Security > Accessibility and allow Terminal (or wherever you run the app from). macOS requires permission to simulate key events.
What's Tony Stark Mode?
It replaces all slap sounds with J.A.R.V.I.S. sci-fi beeps. Double-clap your MacBook and it opens Terminal with Claude Code and plays the Iron Man soundtrack. Yes, really.
Is this on the App Store?
No. IOKit HID access requires running outside the macOS sandbox, which the App Store doesn't allow. You build it from source.
