FAQ

macOS won't let me open it?
Right-click (or Control-click) the app and select "Open" instead of double-clicking. macOS will show a dialog — click "Open" to confirm. You only need to do this once.
Why does it need sudo?
The MacBook's accelerometer requires root privileges to access. The app only reads sensor data locally — nothing leaves your machine.
Which Macs are supported?
Apple Silicon MacBooks with M1 Pro or newer. These have a built-in accelerometer. The base M1, Mac Mini, Mac Studio, and Intel Macs don't have it.
Will it damage my MacBook?
The accelerometer detects vibration — you don't need to hit hard. A firm tap on the side or trackpad area is enough. Slap responsibly.
Can I add my own sounds?
Yes. Select "Custom" in the Sounds tab and add your own .mp3 files.
The key press isn't working?
Go to System Settings > Privacy & Security > Accessibility and allow Terminal. macOS requires permission to simulate key events.
What's Tony Stark Mode?
It replaces all slap sounds with J.A.R.V.I.S. sci-fi beeps. Double-clap your MacBook and it opens Terminal with Claude Code and plays the Iron Man soundtrack.
Is this on the App Store?
No. Accelerometer access requires running outside the macOS sandbox, which the App Store doesn't allow. SlapSound is distributed directly by agaro.ai.
